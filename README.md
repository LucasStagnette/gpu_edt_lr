# gpu_edt_lr LR
 
Projet récupère l'emploi du temps sur GPU, puis récupère toutes les semaines automatiquement toutes les 30 minutes et les concatène dans un seul fichier .ics accessible en ligne pour pouvoir le renseigner sur nos calendriers personnels. 
Ce fichier se met à jour toutes les 30 minutes grâce aux actions Github.
